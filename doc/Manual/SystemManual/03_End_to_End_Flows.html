End-to-End Business Flows

Back to Table of Contents

Order to Cash

  1. Create order in Sales -> Transactions -> Sales Orders.
  2. Approve/release through approval workflow where configured.
  3. Create invoice in Sales -> Transactions -> Sales Invoice.
  4. Post financial impact to customer balances and ledger.
  5. Receive funds in Accounts Receivable -> Transactions -> Receipts.
  6. Apply receipts in Accounts Receivable -> Transactions -> Receipts Allocation.

Procure to Pay

  1. Create purchase order in Purchases -> Transactions -> Purchase's Order.
  2. Approve purchase documents in Document Approvals.
  3. Process receipts and supplier-facing purchase entries.
  4. Enter bills in Accounts Payable -> Transactions -> Enter Bills.
  5. Prepare and approve vouchers in Accounts Payable / Cash Management.
  6. Execute payment and allocate to supplier open entries.

Inventory to Production to Quality

  1. Maintain stock, stores, units, and posting groups in Inventory -> Set Up.
  2. Transfer and issue stock in Inventory -> Transactions.
  3. Run production in Production -> Transactions -> Production.
  4. Record test outcomes in Production -> Transactions -> Production Quality Test.
  5. Complete review in Production -> Transactions -> Review Quality Test.

Record to Report

  1. Post journals in General Ledger -> Transactions.
  2. Run financial statements in General Ledger -> Inquiries and Reports.
  3. Close periods in General Ledger -> Set Up -> Create/Close Financial Periods.
  4. Run depreciation in Fixed Assets -> Transactions -> Depreciation Journal.
